> 每次新建/补充题库前，先对照此表，确保每个知识点都有对应题目。
> **⚠️ 任何数据修改（题库、题面、选项、干扰项、对话台词）必须对照下方「白名单」表，凡出现禁用词一律视为 P1 bug。**

## 🛡️ U1L1 词汇 / 句型 白名单（硬约束）

### ✅ 合法词汇（凡英文内容只能来自以下集合）
- **动物名词**：bear / horse / bird / panda / rabbit / monkey / duck
- **形容词**：big / cute / fast
- **动词**：fly（"It can fly."）/ run（"It can run fast."）/ can / is / are
- **限定词 / 代词**：this / that / it / they / a / the
- **感叹 / 呼唤**：Look!
- **拼写字母**：a-z 用于 coop_spell_word，仅限上述单词的字母

### ❌ 禁用词（出现即 P1 bug）
red / brown / black / white / yellow / orange / green / blue
cat / tiger / dog / lion / elephant
Cool / nice / good / small / slow / tall / short / hot / cold / beautiful
has / have / and / or / but / very / too / so
long / legs / tail / ears / eyes / head / body

### ✅ 合法句型（仅允许以下结构）
- `This is a ___.`
- `That is a ___.`
- `It's ___.` （空格只能填 big / cute / fast 之一）
- `It can fly.`
- `It can run fast.`
- `They are ___s.`
- `Look!` 可作为句首

### 🧪 修改任何 data/u1l1_coop.js 后的自检
1. `grep -nE '\\b(red|brown|black|white|yellow|cat|tiger|dog|Cool|small|slow|beautiful|has|and|long|legs|hot|cold)\\b' data/u1l1_coop.js` 必须 0 命中
2. 新增句型不得超出上方 6 种结构
3. 新词必须先在白名单追加并经用户确认
